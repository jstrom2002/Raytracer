# Raytracer #
### Notes ###
This repository is a port of the source files from Peter Shirley's "Ray Tracing In One Weekend" series of books to a program that uses several OpenGL-related libraries.

## Dependencies ##
Windows  
GLFW3  
GLEW  
OpenGL  
STB
